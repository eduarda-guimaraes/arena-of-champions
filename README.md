# Arena of Champions ⚔️  
### Turn-Based Battle Game

## Module 4 Challenge - Back-end (Java and Object-Oriented Programming)

This project was developed as the **final challenge of Module 4**, focused on **Java and Object-Oriented Programming**.

The goal is to build a **terminal-based RPG battle system**, where the player chooses a hero and fights enemies in a **turn-based combat system** until victory or defeat.

---

## 🎮 Game Overview

In **Arena of Champions**, the player:

- Chooses a **name**
- Selects a **character class**
- Faces enemies in a **turn-based battle**
- Chooses actions such as **attack, defend, or use potions**

The game ends when:

- The player **defeats the enemy**
- The player **loses all health**

---

## 🧠 Concepts Applied

This project applies several Java and OOP concepts:

- Classes and Objects
- Inheritance
- Constructors
- Methods
- Control Flow (`if`, `while`)
- Collections (`List`, `ArrayList`)
- Modularization
- User input with `Scanner`

---

## 🧙 Character Classes

The player can choose between three classes:

| Class | Description |
| :--- | :--- |
| **Warrior** | High health and defense |
| **Mage** | High attack but low defense |
| **Archer** | Balanced attributes |

---

## 🧪 Inventory System

Each character starts with **3 potions**.

- Restore **20 health points**
- Are stored in a **List inventory**
- Can be used during battle

---

## ⚔️ Battle System

The battle works in **turns**:

1. The game displays the status of both characters.
2. The player chooses an action:
   - **Attack**
   - **Defend**
   - **Use Potion**
3. The enemy performs its action.
4. The battle continues until someone is defeated.

# Arena dos Campeões ⚔️  
### Jogo de Batalha por Turnos

## Desafio do Módulo 4 - Back-end (Java e Programação Orientada a Objetos)

Este projeto foi desenvolvido como o **desafio final do Módulo 4**, focado em **Java e Programação Orientada a Objetos**.

O objetivo é construir um **sistema de batalha de RPG baseado em terminal**, onde o jogador escolhe um herói e luta contra inimigos em um **sistema de combate por turnos** até a vitória ou derrota.

---

## 🎮 Visão Geral do Jogo

Em **Arena dos Campeões**, o jogador:

- Escolhe um **nome**
- Seleciona uma **classe de personagem**
- Enfrenta inimigos em uma **batalha por turnos**
- Escolhe ações como **atacar, defender ou usar poções**

O jogo termina quando:

- O jogador **derrota o inimigo**
- O jogador **perde toda a vida**

---

## 🧠 Conceitos Aplicados

Este projeto aplica diversos conceitos de Java e POO:

- Classes e Objetos
- Herança
- Construtores
- Métodos
- Estruturas de Controle (`if`, `while`)
- Coleções (`List`, `ArrayList`)
- Modularização
- Entrada de dados com `Scanner`

---

## 🧙 Classes de Personagem

O jogador pode escolher entre três classes:

| Classe | Descrição |
| :--- | :--- |
| **Guerreiro** | Alta vida e defesa |
| **Mago** | Alto ataque, mas baixa defesa |
| **Arqueiro** | Atributos equilibrados |

---

## 🧪 Sistema de Inventário

Cada personagem começa com **3 poções**.

- Recuperam **20 pontos de vida**
- São armazenadas em um **inventário do tipo List**
- Podem ser usadas durante a batalha

---

## ⚔️ Sistema de Batalha

A batalha funciona em **turnos**:

1. O jogo exibe o status de ambos os personagens.
2. O jogador escolhe uma ação:
   - **Atacar**
   - **Defender**
   - **Usar Poção**
3. O inimigo executa sua ação.
4. A batalha continua até que alguém seja derrotado.
